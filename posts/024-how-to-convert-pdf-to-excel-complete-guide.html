Pro Tips for Best Results

  • Clean your data first - Remove extra headers and footers
  • Check table structure - PDF tables with merged cells may need adjustment
  • Use OCR for scanned PDFs - Scanned documents need OCR before conversion
  • Preview before converting - Check that tables are detected correctly
  • Verify the output - Open the Excel file and spot-check a few rows

Common Issues and Solutions

Issue Solution
Tables not detected Switch to Manual mode and select tables

Merged cells broken Adjust formatting in Excel after conversion

Scanned document Use OCR mode first
