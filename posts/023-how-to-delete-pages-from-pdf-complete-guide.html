How to Delete Pages from PDF - Complete Guide

Sometimes your PDF has pages you don't need. Maybe it's a blank page that snuck in. Maybe you scanned extra pages by mistake. Or perhaps you want to remove confidential information before sharing.

Whatever the reason, deleting pages from a PDF is easy. This guide shows you how.

Why Delete Pages from PDF?

Reason Example
Remove blanks Delete empty pages that waste space
Fix scanning errors Remove duplicate or mis-scanned pages
Protect privacy Delete sensitive information before sharing
Shorten documents Keep only the pages you need
Improve file size Fewer pages = smaller file
Prepare for sharing Send only relevant sections

Delete vs Extract: What's the Difference?

Feature Delete Pages Extract Pages
What it does Removes unwanted pages Saves selected pages as new file
Original file Becomes smaller Remains unchanged
Output Same file, fewer pages New file with selected pages
Best for Removing mistakes Saving specific pages

Example:

  • Delete: You have a 20-page PDF, remove pages 3-5, now it's 17 pages.
  • Extract: You have a 20-page PDF, save pages 3-5 as a new 3-page PDF.

Pro Tips for Best Results

  • Preview before deleting - Double-check you're removing the right pages
  • Work on a copy - Keep the original file in case you need it
  • Use page ranges - Faster than selecting individual pages
  • Check page numbers - PDF page numbers may differ from printed page numbers
  • Delete blanks first - Then compress the file for even smaller size

Common Scenarios

Scenario 1: Delete Blank Pages

Scenario 2: Remove Duplicate Pages

Sometimes scanning creates duplicates. Delete the extra copies.

Scenario 3: Remove Sensitive Content

Scenario 4: Shorten a Document

You only need pages 10-20 from a long report. Delete everything else.

Scenario 5: Prepare for Printing

Remove title pages, cover pages, or blank backs to save paper.
